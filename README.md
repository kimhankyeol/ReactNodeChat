# ReactNodeChat
React+Socket.io+Node


React(채팅 UI)+ SocketIO + nodejs (채팅 서버)
React(Socket client)에서 socket.emit 하면  node.js(Socket server)에서 socket.on 해서 요청된 이벤트를 실행



실행 

1.React 
cd frontend 
npm install  or yarn add 
npm start or yarn start

2 Node JS
cd socket-backend
npm install or yarn add
npm start or yarn start
